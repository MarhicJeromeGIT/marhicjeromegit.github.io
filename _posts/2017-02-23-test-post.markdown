# HELLO BLOG

Sometime, the need arise to pass extra arguments to a function. Something like this works fine:

```ruby
def foo(options)
  default = {a: 1, b: 1}
  params = default.merge(options)
end
```

However it lacks (...). Since version 2.0 Ruby allows for keywords arguments, that is instead of writing

```ruby
def foo(options)
  a = options[:a]
end
```

You can write

```ruby
def foo(a:)
  # we already have a
end
```
Let's use keywords arguments to enable a mix of default and extra parameters for our functions !

```ruby
# notice the double splat '**' 
def foo(a: 1, b: 1, **extra_key) 
  params = method(__method__).parameters
  params.pop # remove the 'extra_keywords' element
  default_params = (params.map{|key, arg| [arg, (binding.local_variable_get arg)]})
  options = default_params.merge(extra_key)
  # Now options is the merge of {a: 1, b:1} and the extra_key params

def foo(a: 1, b: 1, **extra_keys)
  params = method(__method__).parameters
  params.pop
  default_params = (params.map{|key, arg| [arg, (binding.local_variable_get arg)]})
  puts default_params
  default_params = Hash[default_params]     
  puts default_params
  options = default_params.merge(extra_keys)
  return options
end
```
